date: 2026-03-29
summary: Static export keeps this writing site honest by forcing content, metadata, routes, and deployment failures to resolve at build time.
---

Static export removes the temptation to solve content problems with runtime infrastructure.

## Constraint as design input

For a writing-first site, that is a feature:

- deployment becomes simple
- failures happen at build time
- content stays portable
